# bring-global-salary

Feature "Traer el sueldo": conectar otro banco, detectar cuando llega el salario y traerlo automáticamente. Versión multi-país (agnóstica y configurable por país).

## Cómo correr el proyecto

- **No necesitas cuenta de Expo.** Puedes usar Expo Go sin iniciar sesión.
- Instala [Expo Go](https://expo.dev/go) en tu celular o usa el simulador.

```bash
npm install
npm start
```

Luego escanea el QR con Expo Go (mismo WiFi) o pulsa `i` (iOS) / `a` (Android) en la terminal si usas simulador.

## Scripts

| Comando     | Descripción              |
|------------|--------------------------|
| `npm start` | Inicia el servidor Expo  |
| `npm run ios` | Abre en simulador iOS   |
| `npm run android` | Abre en emulador Android |
